QA Automation test

почту;
2) С помощью Selenium определить, сколько нашлось писем;

количество писем. Указать тему письма "Тестовое задание. <Фамилия>", где
<Фамилия> - это Ваша фамилия;

1) Java, Selenium Webdriver;
1) Selenium GRID. Для запуска сервера перейти в консоли в папку resources, выполнить:
 java -Dwebdriver.chrome.driver=chromedriver.exe -jar selenium-server-standalone-3.141.59.jar
 Открыть в браузере страницу:
 http://localhost:4444/wd/hub/static/resource/hub.html

3) Реализовано формирование отчетов о пройденных тестах через Allure.
